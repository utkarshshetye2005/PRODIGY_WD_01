# 📌 Navigation Menu

A simple and responsive Navigation Menu built using HTML, CSS, and JavaScript. This project demonstrates how to create an interactive and user-friendly navigation bar for websites.

---

## 🚀 Features

- 📌 Responsive navigation bar  
- 🎯 Smooth scrolling between sections  
- 🎨 Clean and modern UI design  
- 📱 Mobile-friendly layout  
- ⚡ Easy to customize  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
